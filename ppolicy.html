VPN App ကိုယ်ရေးကိုယ်တာမူဝါဒ (Privacy Policy)

1. VPN သုံးစွဲသူအချက်အလက်စုဆောင်းမှု

  • ဤအက်ပ်သည် သုံးစွဲသူ၏ကိုယ်ရေးကိုယ်တာအချက်အလက်များကို အမည်, အီးမေးလ်, ဖုန်းနံပါတ် (လိုအပ်ပါက) များကို တရားဝင်ရည်ရွယ်ချက်များအတွက်သာ စုဆောင်းပါသည်။
  • သင့်အင်တာနက်အသုံးပြုမှု (ဥပမာ: IP Address, Server Location) မသိမ်းဆည်းပါ။
  • VPN App သည် သုံးစွဲသူ၏ Traffic Logs, DNS Queries, သို့မဟုတ် Browsing Data များကို မသိမ်းဆည်းပါ။

2. သုံးစွဲသူအချက်အလက်များအသုံးပြုမှု

  • သုံးစွဲသူများအား VPN Service ၏အရည်အသွေးမြှင့်တင်ရေးနှင့် တိုးတက်မှုအတွက်သာ သင်၏ဒေတာများကို အသုံးပြုမည်။
  • သုံးစွဲသူ၏ ခွင့်ပြုချက် မရှိဘဲ ဒေတာများကို တတိယဖက်များနှင့် မမျှဝေပါ။

3. လုံခြုံရေးနည်းလမ်းများ

  • သင့်ဒေတာများအား AES-256 Encryption, SSL/TLS Security Protocols တို့ဖြင့် ကာကွယ်ထားပါသည်။
  • VPN Server များတွင် No-Logs Policy (ဒေတာမသိမ်းဆည်းမှုမူဝါဒ) ရှိပါသည်။

4. တတိယဖက်ဝန်ဆောင်မှုများ

  • တတိယဖက်ဝန်ဆောင်မှုများ၏ မူဝါဒများကိုလည်း စိစစ်ဖတ်ရှုရန် သုံးစွဲသူများကို တိုက်တွန်းပါသည်။
