ကိုယ်ရေးကိုယ်တာမူဝါဒ

ဤ VPN အက်ပ်သည် သင့်ကိုယ်ရေးကိုယ်တာအချက်အလက်များကို လုံခြုံစွာသိမ်းဆည်းထားမည်ဖြစ်ပြီး သင့်ခွင့်ပြုချက်မရှိဘဲ မည်သည့်ဒေတာကိုမျှဝေမည်မဟုတ်ပါ။

1. VPN သုံးစွဲသူအချက်အလက်စုဆောင်းမှု

- ဤအက်ပ်သည် သုံးစွဲသူ၏ကိုယ်ရေးကိုယ်တာအချက်အလက်များကို **အမည်**, **အီးမေးလ်**, **ဖုန်းနံပါတ်** (လိုအပ်ပါက) စုဆောင်းနိုင်သည်။

- သင့်အင်တာနက်အသုံးပြုမှု (ဥပမာ: IP Address, Server Location) **မသိမ်းဆည်းပါ**။

2. သုံးစွဲသူအချက်အလက်များအသုံးပြုမှု

- VPN Service ၏အရည်အသွေးမြှင့်တင်ရေးနှင့် တိုးတက်မှုအတွက်သာ သင်၏ဒေတာများကို အသုံးပြုမည်။

- သင့်ခွင့်ပြုချက်မရှိဘဲ ဒေတာများကို တတိယဖက်များနှင့် မမျှဝေပါ။

3. လုံခြုံရေးနည်းလမ်းများ

- သင့်ဒေတာများအား **AES-256 Encryption**, **SSL/TLS Security Protocols** တို့ဖြင့် ကာကွယ်ထားသည်။

- VPN Server များတွင် **No-Logs Policy** (ဒေတာမသိမ်းဆည်းမှုမူဝါဒ) ရှိသည်။
